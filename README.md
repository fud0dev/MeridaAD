# ⚽ ADMérida Info · Mérida XYZ

> **Dashboard deportivo premium del equipo de la Ciudad Eterna.**

**ADMérida Info** es un panel de control interactivo diseñado para el seguimiento en tiempo real del AD Mérida. El proyecto ofrece una experiencia visual premium, centrada en los resultados, la clasificación y el calendario de la competición (Primera RFEF), integrándose dentro del ecosistema **Mérida XYZ**.

## 🚀 Características Principales

- **Resultados en Vivo**: Sistema de rastreo automatizado para actualizar marcadores y estados de partidos.
- **Clasificación Dinámica**: Visualización clara y actualizada de la tabla clasificatoria.
- **Estética High-Definition**: Integración de logotipos de clubes en alta definición y una interfaz diseñada bajo el lenguaje "Premium Light".
- **Sincronización Inteligente**: Datos actualizados cada 2 horas para garantizar la relevancia de la información deportiva.

## 🛠️ Tecnologías Utilizadas

- **Scraper Core**: Python con librerías de procesamiento para la extracción de datos deportivos de fuentes externas.
- **Frontend Layer**: HTML5, CSS3 (Custom Properties) y JavaScript para el renderizado dinámico de la tabla y resultados.
- **Automation Pipeline**: GitHub Actions para mantener el dashboard sincronizado automáticamente con la realidad competitiva.

## 📊 Fuentes de Datos

La información se obtiene de proveedores de datos deportivos públicos, procesada para ofrecer una interfaz limpia y libre de publicidad, centrada exclusivamente en la utilidad para el aficionado del AD Mérida.
